# Project Summary
This project is a web application that supports researchers in managing their academic literature and enhancing their research workflow. It integrates features for defining reading purposes, logging search records, managing notes, and visualizing relationships between topics. Recent updates have introduced advanced functionalities such as workspace analytics, enhanced research content visualization, and a pre-writing block, all designed to improve user productivity and research output.

# Project Module Description
- **Purpose Workspace**: Define reading purposes with custom inputs for premium users.
- **Entry Paper Workspace**: Add search records, keywords, and candidate papers; mark relevance and track discovery paths.
- **Read Workspace**: Read and annotate papers; manage literature and permanent notes.
- **Expand Workspace**: Manage references, cited papers, and similar topics, with manual and auto-identification options.
- **Visualize Workspace**: 
  - **Workspace Analytics**: Overview stats, artifact breakdown, and project insights.
  - **Research Content Visualization**: 
    - **Papers Overview**: View and sort all papers with annotation/note counts.
    - **External Files**: Upload files with an "Add to Visual Artifacts" option.
    - **AI Summarize Notes (Premium)**: AI-powered note summarization.
    - **Permanent Notes**: Create and manage permanent notes.
    - **Synthesis Tables**: Multi-table synthesis with AI auto-fill.
  - **Visualization Tools**: Directory of common research visualization tools with options to add custom tools.
- **Draft Workspace**: Draft research papers with structured checks for content and citations, including:
  - **Project Switcher**: Dropdown for switching between projects, displaying titles, goals, and progress.
  - **Draft Page Layout**: Three-column layout for managing artifacts, writing blocks with reporting style selectors, and structure checks.
  - **Pre-Writing Block**: Strategies for brainstorming, listing, clustering, and free writing.

# Usage
1. **Installation**: 
   - Install dependencies using the package manager (e.g., `pnpm install`).
2. **Build**: 
   - Build the application using the command (`pnpm run build`).
3. **Run**: 
   - Start the development server using the command (`pnpm run dev`).
